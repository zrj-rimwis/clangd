Build time overheads.

Test setup:
vanilla DragonFly BSD master, no modifications, standard opt level "-pipe -O"
Old Lenovo E540 laptop, 16GB of RAM, Haswell cpu, 4 cores + HT (8 threads)
CPU: Intel(R) Core(TM) i7-4712MQ CPU @ 2.30GHz (2294.70-MHz K8-class CPU)
turbo disabled (for more reproducible build times)
All compilations are done on tmpfs mount, "time make -j8" in gnu/usr.bin/clung.

du -hsc contrib : 65M (2338 sources)
make -j8 depend : ~2.5min
make -j8        : 19min 15s
du -hsc .       : 571M
 clang          : 52M
 LLVM.gold      : 28M

add twice that to the reference /usr/src base system compilation,
that bootstraps two compilers gcc47 and gcc50 already:
